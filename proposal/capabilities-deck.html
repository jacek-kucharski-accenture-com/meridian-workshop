Required · R1

Reports Module Remediation

We audit before we fix. No guessing at what's broken.

1

Audit

Map every filter against the data flow. Identify where query params are missing, mismatched, or ignored.

2

Reproduce

Run the application, trigger each failure mode, document defect-by-defect with expected vs. actual behavior.

3

Remediate + Test

Fix in order of user impact. Each fix ships with a Playwright test — no fix without coverage.

Assumption: Meridian's partial notes are a starting point. Our audit is the authoritative list — we fix everything we find.
