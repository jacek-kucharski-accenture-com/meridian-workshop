# Pricing

**RFP reference:** §4.5
**Structure:** Fixed-fee, not-to-exceed $100,000

---

## Fee Schedule by Phase

| Phase | Scope | Fixed Fee |
|---|---|---|
| Phase 0 — Orientation | Environment setup, architecture review, Reports audit | $8,000 |
| Phase 1 — Remediation | R1 Reports fixes + R3 baseline test coverage | $22,000 |
| Phase 2 — Restocking | R2 Restocking view + R3 extended tests + R4 architecture doc | $38,000 |
| Phase 3 — Desired items + Handoff | D1 UI refresh, D2 i18n, D3 dark mode, final handoff | $28,000 |
| **Total** | **All required + desired items** | **$96,000** |

Total is within the $100,000 not-to-exceed envelope, with $4,000 held as contingency.

---

## Required Items Only

If Meridian elects to scope only R1–R4 (no desired items), the fee is:

| Scope | Fee |
|---|---|
| Phases 0–2 only (R1, R2, R3, R4) | $68,000 |

Desired items (D1–D3) can be added in a Phase 3 change order at the rates below.

---

## Desired Item Add-Ons (if scoped separately)

| Item | Fee |
|---|---|
| D1 — UI modernization | $10,000 |
| D2 — i18n extension (Tokyo views, Japanese) | $8,000 |
| D3 — Dark mode | $6,000 |
| D1 + D2 + D3 (bundled) | $22,000 |

---

## Payment Terms

| Milestone | Amount due |
|---|---|
| Contract signature | 25% ($24,000 on full scope) |
| Phase 1 completion (end of Week 3) | 25% |
| Phase 2 completion (end of Week 6) | 30% |
| Phase 3 completion + handoff | 20% |

---

## Pricing Assumptions

| # | Assumption |
|---|---|
| P1 | Engagement is fixed-fee; hours are not tracked or billed separately |
| P2 | Data layer remains JSON files — no database procurement, licensing, or migration costs |
| P3 | Deployment is out of scope — we deliver to development environment; Meridian IT handles production |
| P4 | Japanese translations for D2 are provided by our team for Tokyo-priority views; additional languages billed separately |
| P5 | Travel is not required — engagement is fully remote |
| P6 | Meridian provides timely access to any partial issue notes and availability for two-week checkpoint demos |
| P7 | Scope changes after contract signature are handled via written change orders; minor adjustments within existing scope are included |

---

## What This Does Not Include

- Database infrastructure or licensing
- Production deployment or hosting changes
- Additional language translations beyond Japanese (D2)
- Post-handoff support or maintenance (available under a separate retainer)
